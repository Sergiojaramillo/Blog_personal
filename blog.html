Iot en la agricultura de presicion

Actualmente existe un auge de herramientas que simplifiquen la obtención, almacenamiento y procesamiento de datos climatologicos; estos datos pueden ayudar a no comprometer los ciclos productivos en los terrenos de siembra, siendo esta una de las razones principales para utilizar la agricultura de precisión. Los factores mas influyentes logran ser la condición de resequedad y la alta temperatura por radiacion solar. Para ello es necesario un sistema que permita integrar de manera sencilla y economica la visualizacion de variables y posibles estados del clima para la toma de diferentes desiciones en la siembra y potsiembra. Normalmente para crear un dominio web se requiere tanto de infraestructura y equipo especializado ya sean servidores que requiere de un alto costo para adquirirlos, además de la utilización de diferentes herramientas de programación y lenguajes que, estar familiarizado con estos resulta muy inconveniente a la hora de generar un servicio web en un corto plazo. La utilización de hardware libre permite generar un sistema economico y de facil utilización como es el Arduino YUN para la adquisición apropiada y en tiempo real de las variables meteorologicas que se encuentran directamente relacionadas a la hora de tomar una decisión en un proceso agronomico, además de la utilización de software libre como lo es el framework Sparkjava para la creación de un servicio web local de manera rapida con un lenguaje sencillo de programación y sin necesidad de equipos costosos; permitirá realizar el monitoreo de las condiciones meteorologicas presentes en un cultivo adem{as de la integración con aplicaciones en la nube que permitirán procesar estos datos obtenidos y realizar pronosticos y recomendaciones que facilitarán la toma de decisiones
