Inauguración de Parkeando

Parkeando es una aplicación mobile disponible para Android (próximamente para iOS), la cual permite encontrar un parqueadero fácilmente, además de escoger entre las diferentes características que estos tienen.

Sistema IoT en la agricultura de presición

Esta tesis de aplicación pretende fomentar la interoperabilidad de hardware y software libre, al utilizar una tarjeta Arduino YUN y sensores especiales generando un sistema de Internet de las Cosas, para la adquisición de variables meteorologicas en un entorno de agricultura de Precisión.
